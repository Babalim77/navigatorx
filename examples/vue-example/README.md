# NavigatorX - Vue 3 Example

Demonstration of NavigatorX keyboard shortcut manager with Vue 3, Composition API, and Vue Router.

## Features

- ✅ Global keyboard shortcuts
- ✅ Vue Router integration
- ✅ Composition API with `useNavigatorX` composable
- ✅ Search modal (Shift+K)
- ✅ Help dialog (Shift+?)
- ✅ Page-specific shortcuts
- ✅ TypeScript support
- ✅ Teleport for modals

## Quick Start

```bash
npm install
npm run dev
```
