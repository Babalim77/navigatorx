# NavigatorX - React + Vite Example

Demonstration of NavigatorX keyboard shortcut manager with React, Vite, and React Router.

## Features

- ✅ Global keyboard shortcuts
- ✅ React Router integration
- ✅ Custom hooks (`useNavigatorX`)
- ✅ Search modal (Shift+K)
- ✅ Help dialog (Shift+?)
- ✅ Page-specific shortcuts
- ✅ TypeScript support

## Quick Start
```bash
npm install
npm run dev
```
